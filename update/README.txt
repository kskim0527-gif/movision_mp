================================================================
 movision BLE OTA 업데이트 도구  사용 안내
================================================================

【 실행 방법 】
  "BLE_업데이트.bat" 파일을 더블클릭하면 자동으로 실행됩니다.

【 사전 조건 】
  1. PC의 블루투스가 켜져 있어야 합니다
  2. movision 장치 전원이 ON 상태여야 합니다
  3. Python 3.8 이상이 설치되어 있어야 합니다
     → https://python.org
  4. ESP-IDF로 빌드가 완료되어 있어야 합니다
     → ..\build\movision.bin   (펌웨어)
     → ..\build\storage.bin   (이미지/GIF 파일)

【 업데이트 절차 】
  1) BLE_업데이트.bat 더블클릭
  2) bleak 라이브러리 자동 설치 (최초 1회)
  3) 업데이트 종류 선택 (펌웨어/Storage/둘다)
  4) 주변 BLE 장치 자동 스캔
  5) movision 장치 선택 (★ 표시)
  6) 파일 자동 전송 (진행률 표시)
  7) 장치 자동 재시작

【 포함 파일 】
  BLE_업데이트.bat  ← 이 파일을 실행
  ble_ota.py        ← 파이썬 업데이트 스크립트

【 문제 해결 】
  - "OTA 서비스를 찾지 못했습니다"
    → 장치가 이미 연결된 상태이면 연결을 끊고 재시도

  - 전송 속도가 느린 경우
    → 장치를 PC에 가까이 두세요 (1m 이내 권장)

  - bleak 설치 오류
    → python -m pip install bleak  명령을 수동으로 실행
